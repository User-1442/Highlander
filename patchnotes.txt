  ALPHA VERSIONS
------------------
Alpha 1.0: Added SSR to the game engine. It worked bad at first being grainy and inaccurate.
Alpha 1.1_S1: Added basic changeable textures to the engine. Not object wise yet and also not PBR yet. But holy it took a full month because I was busy with other stuff!
Alpha 1.1_S2: Made the textures object wise which oddly broke the orientation of the textures. Will continue to fix
Alpha 1.1: Added object wise material support that also behaves correctly around textures.
Alpha 1.2: Added a save system so scenes aren’t reset when you turn off the game engine. You can actually make a scene now!
Alpha 1.2.1: Slightly modified the SSR by adding a blur pass to make it slightly less grainy and easier to look at. Also added Windows support with DLL’s and EXE’s.
Alpha 1.2.2: Overhauled the SSR to make the entire game engine up to 4x faster in optimal conditions. Windows support speed greatly increased. Switched from mipmap blur to gaussian blur to make it more sophisticated. Results with the blur vary.
Alpha 1.2.3: Modified the SSR again to completely remove the grain. Revamped the UI with a splash screen when you open up Highlander and changed project selection so that instead of typing in a file you can just click a button and load a project. Added graphics settings to modify the load of the SSR and blur. Added resolution settings and vsync settings. Moved controls around in the UI and added the ability to copy/paste objects with CTRL C and CTRL V.
Alpha 1.2.3_P1: Fixed a bug where you couldn’t change any textures
